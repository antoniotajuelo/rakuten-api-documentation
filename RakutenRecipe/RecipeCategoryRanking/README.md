
# Recipe/CategoryRanking

## Description

Gets the top 4 ranked recipes.
## Resource URL

https://app.rakuten.co.jp/services/api/Recipe/CategoryRanking/20170426
## Resource Information

* **Auth Type** App Key
* **Affiliation Support** No
