
# Travel/KeywordHotelSearch

## Description

Gets up to 30 hotels by keyword.
## Resource URL

https://app.rakuten.co.jp/services/api/Travel/KeywordHotelSearch/20131024
## Resource Information

* **Auth Type** App Key
* **Affiliation Support** Yes
