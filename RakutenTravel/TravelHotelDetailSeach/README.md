
# Travel/HotelDetailSeach

## Description

Gets a hotel details by hotel number.
## Resource URL

https://app.rakuten.co.jp/services/api/Travel/HotelDetailSearch/20131024
## Resource Information

* **Auth Type** App Key
* **Affiliation Support** Yes
