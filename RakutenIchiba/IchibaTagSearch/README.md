
# IchibaTag/Search

## Description

Gets up to 10 tags and their tag group by tag id.
## Resource URL

https://app.rakuten.co.jp/services/api/IchibaTag/Search/20140222
## Resource Information

* **Auth Type** App Key
* **Affiliation Support** No
