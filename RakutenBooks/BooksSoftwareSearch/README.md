
# BooksSoftware/Search

## Description

Gets up to 30 software items by title, OS, maker, label, JAN or genre.
## Resource URL

https://app.rakuten.co.jp/services/api/BooksSoftware/Search/20170404
## Resource Information

* **Auth Type** App Key
* **Affiliation Support** Yes

## Parameters
